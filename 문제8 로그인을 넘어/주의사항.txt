어떤 쿠키가 꼭 있어야 하나?

크롬/엣지(Chromium)에서 쿠키 복사하는 법
네이버에 로그인한 상태에서 F12 → Network 탭 열기.
필터링

필터박스에 naver.com 입력

페이지 새로고침(F5) 해서 요청이 뜨면, 목록에서 https://www.naver.com/ (Type: document) 항목을 클릭.

오른쪽 사이드에서 Headers 탭 → Request Headers 섹션으로 스크롤.

Cookie 항목의 “값(value)만” 드래그 복사하세요.

주의: Cookie: 라벨까지 포함하면 안 됩니다. NID_SES=...; NID_AUT=...; ... 이런 오른쪽 값만 가져오세요.

붙여넣을 때는 제가 만든 스크립트의 “쿠키 문자열 입력” 프롬프트에 그대로 붙이면 됩니다.
스크립트는 a=1; b=2; ... 형태를 파싱해서 세션 쿠키로 넣어요.


네이버는 자주 바뀌지만 보통 NID_SES, NID_AUT 같은 로그인 관련 쿠키들이 포함됨
따라서 전체 Cookie 값을 통째로 가져오는 게 안전합니다(하나씩 고르지 말고 전부).

파이어폭스일 때

로그인 상태에서 F12 → 네트워크 → 새로고침.

https://www.naver.com/ 문서 요청 선택.

요청 헤더(Headers)에서 Cookie 값만 복사.

대안(조금 더 정확한 방법)

DevTools 상단의 Application(크롬)/Storage(파폭) 탭 → Cookies → https://www.naver.com
 선택 → 오른쪽 표에서 Name=Value들을 확인할 수 있어요.
이 경우 표 전체를 복사→ Name=Value; Name2=Value2; ... 형태로 직접 이어 붙이기 하면 동일하게 쓸 수 있습니다.
